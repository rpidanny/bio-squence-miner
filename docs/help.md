=============

* [`darwin help [COMMAND]`](#darwin-help-command)

## `darwin help [COMMAND]`

Display help for darwin.

```
USAGE
  $ darwin help [COMMAND...] [-n]

ARGUMENTS
  COMMAND...  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for darwin.
```
