# Usage

<!-- usage -->

```sh-session
$ npm install -g @rpidanny/darwin
$ darwin COMMAND
running command...
$ darwin (--version)
@rpidanny/darwin/0.0.0 darwin-arm64 node-v20.8.1
$ darwin --help [COMMAND]
USAGE
  $ darwin COMMAND
...
```
